# Curso Maestro de Responsive Web Design

Bienvenido al repositorio definitivo para aprender Diseño Web Responsivo. Este curso está diseñado para llevarte desde los fundamentos históricos hasta la implementación de layouts complejos modernos.

## Contenido del Curso

### 1. Documentación Teórica (`docs/`)
Lee estos archivos en orden para construir una base sólida.

- **[01. Origen y Filosofía](docs/01-origen-y-filosofia.md)**: Historia, evolución y la importancia del enfoque Mobile First.
- **[02. Fundamentos Técnicos](docs/02-fundamentos-tecnicos.md)**: El Viewport, Media Queries y Unidades Relativas.
- **[03. Sistemas de Layout](docs/03-sistemas-de-layout.md)**: Guía profunda sobre Flexbox y CSS Grid.
- **[04. Imágenes y Media](docs/04-imagenes-y-media.md)**: Cómo manejar imágenes, videos y la etiqueta picture.

### 2. Ejemplos Prácticos (`examples/`)
Código funcional que demuestra los conceptos aprendidos.

- **[01. Hola Media Queries](examples/01-hola-media-queries/)**: Tu primer contacto con el cambio de estilos según la pantalla.
- **[02. Tarjeta Mobile First](examples/02-tarjeta-mobile-first/)**: Creación de un componente que evoluciona de móvil a desktop.
- **[03. Layout Completo](examples/03-layout-completo/)**: Una landing page completa con menú hamburguesa y grid.

### 3. Ejercicios (`exercises/`)
Pon a prueba tus habilidades con estos retos.

- **[Instrucciones Generales](exercises/README.md)**
- **[01. Flexbox Nav](exercises/01-flexbox-nav/)**: Crea una barra de navegación responsiva.
- **[02. Grid Gallery](exercises/02-grid-gallery/)**: Crea una galería automática sin media queries.
- **[03. Responsive Card](exercises/03-responsive-card/)**: Transforma una tarjeta de perfil radicalmente.

## Cómo usar este repositorio

1.  Clona el repositorio o descarga los archivos.
2.  Abre la carpeta en tu editor de código favorito (VS Code recomendado).
3.  Lee la teoría en la carpeta `docs`.
4.  Analiza el código en `examples`.
5.  Intenta resolver los retos en `exercises` escribiendo tu propio CSS.

¡Disfruta el aprendizaje!
