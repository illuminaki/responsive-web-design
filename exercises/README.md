# Ejercicios Prácticos

La mejor forma de aprender es haciendo. Aquí encontrarás tres retos diseñados para poner a prueba tus conocimientos sobre Responsive Design, Flexbox y Grid.

## Instrucciones Generales

1.  Cada carpeta contiene un archivo `index.html` con la estructura base y un `style.css` (que puede estar vacío o incompleto).
2.  Tu objetivo es escribir el CSS necesario para cumplir con los requerimientos del reto.
3.  ¡No hagas trampa! Intenta resolverlo sin mirar las soluciones (aunque no incluimos soluciones aquí, la idea es que experimentes).

## Los Retos

### 01. Navegación Flexible (Flexbox)
**Objetivo**: Crear una barra de navegación clásica.
- **Móvil**: Los enlaces deben apilarse verticalmente.
- **Desktop**: Los enlaces deben estar en fila horizontal, con el logo a la izquierda y los links a la derecha.

### 02. Galería Automática (Grid)
**Objetivo**: Una galería de imágenes que se adapta sola.
- **Requerimiento**: Usar `auto-fit` y `minmax` para que las columnas se creen automáticamente según el espacio disponible.
- No debes usar Media Queries para el layout de la grilla.

### 03. Tarjeta de Perfil (Layout Complejo)
**Objetivo**: Un componente que cambia drásticamente.
- **Móvil**: Imagen arriba (círculo), texto abajo centrado.
- **Desktop**: Imagen a la izquierda (cuadrado grande), texto a la derecha alineado al inicio.
